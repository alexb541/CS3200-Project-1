# CS3200-Project-1
Designing a relational database using DB browser and sqlite

This project closely models FRC (FIRST Robotics Competition), using a mix of real data from different competitions (extracted by Claude from csv files and researched online) and fabricated data for a mix of attributes that could be applied directly to FRC and more experimental attributes (generated using Claude). Please view everything in the Test Data folder for more information on how data was generated.
